# Jogo Interativo — EU GANHEI!

Um mini game desenvolvido com HTML, CSS e JavaScript puro, onde o botão “Não” foge do usuário e, ao clicar em “Sim”, uma celebração com fogos de artifício toma conta da tela.

---

## Sobre o Projeto

Este projeto foi criado com o objetivo de praticar:

- Manipulação do DOM
- Eventos em JavaScript
- Animações com `requestAnimationFrame`
- Uso da Canvas API
- Programação Orientada a Objetos (POO)
- Estilização com CSS3

A proposta é simples: tentar clicar no botão "Não". Porém, ele se move pela tela para dificultar a ação do usuário.

---

## Funcionalidades

- O botão “Não” se move para posições aleatórias ao passar o mouse.
- O botão “Sim”, quando clicado:
  - Esconde os botões
  - Ativa modo escuro
  - Exibe a mensagem “EU GANHEI!”
  - Inicia uma animação infinita de fogos de artifício
  - Reproduz som de explosão

---

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Canvas API  
- Áudio externo (SoundJay)
